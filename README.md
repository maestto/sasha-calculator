# Sasha Calculator

мобильный PWA-калькулятор зарплаты по сменам

Приложение работает полностью локально: без сервера, без API, с сохранением данных на устройстве

- Быстрый ввод кассы одной рукой через нижнюю фиксированную панель
- Постоянно видимые итоги: смены, касса, зарплата
- Гибкие процентные правила и фикс за смену
- Сохранение данных между запусками
- Экспорт в TXT, CSV и Telegram
- Установка на телефон как PWA и работа офлайн
